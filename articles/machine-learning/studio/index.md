# <a name="azure-machine-learning-studio-documentation"></a>Azure Machine Learning Studio-Dokumentation

Azure Machine Learning Studio ist ein Tool für die Zusammenarbeit per Drag & Drop, mit der Sie Lösungen für Vorhersageanalysen erstellen, testen und bereitstellen können, die mit Ihren Daten arbeiten.
Anhand von Tutorials, Videos und Beispielmodellen wird gezeigt, wie Sie Machine Learning-Modelle mit Studio erstellen und bereitstellen.
