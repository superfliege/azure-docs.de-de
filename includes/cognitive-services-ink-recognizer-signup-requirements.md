Sie benötigen ein [Cognitive Services-API-Konto](../articles/cognitive-services/cognitive-services-apis-create-account.md) mit Zugriff auf die Freihanderkennungs-API. Falls Sie nicht über ein Azure-Abonnement verfügen, können Sie ein kostenloses [Konto erstellen](https://azure.microsoft.com/try/cognitive-services/). Sie können Ihren Abonnementschlüssel über das [Azure-Portal](../articles/cognitive-services/cognitive-services-apis-create-account.md#access-your-resource) abrufen, nachdem Sie Ihr Konto erstellt haben, oder über die [Azure-Website](https://azure.microsoft.com/try/cognitive-services/my-apis), nachdem Sie eine kostenlose Testversion aktiviert haben.
